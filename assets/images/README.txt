Place 2D sprite images for the web game in this directory. Suggested files include:
- background.png (tiling or tall image for the scrolling backdrop)
- player.png (hero craft – red and white panels with metallic highlights)
- enemy.png (standard fighters – blue and black hull plating)
- boss/ (optional folder containing segmented boss sprites)
- bullet.png or muzzle flash sheets

For crisp rendering on high-DPR screens, consider providing images at 2x or 3x the logical canvas size. The in-engine defaults
generate Star Wars-inspired metallic fuselages procedurally, but replacing them with painted sprites can add more detail. All
assets should be licensed for redistribution and commercial use if necessary.
